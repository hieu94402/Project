one shape factor value corresponds to one temperature profile illustrated in part1.py, when the max temperature parameter is changed, the temperature profile moves up/down translationally. Origin is used to generate a graph based on .csv file. The lines with one identical color in figure 1a and 1b should imply one definition of temperature profile.
